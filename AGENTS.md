# Project Notes

This repository is a static GitHub Pages wedding landing page.

## Working Style

- Keep the site simple: plain `index.html` and `styles.css`.
- Do not add a build step unless the user asks for one.
- Preserve the elegant olive wedding style inspired by the Felicity Olive reference.
- Make responsive changes for both mobile and desktop.
- After local edits, commit and push to `main` so GitHub Pages updates automatically.

## Review Guidelines

- Check that links, section anchors, and RSVP email links still work.
- Check that text remains readable on mobile.
- Avoid adding real private wedding details unless the user provides them.
