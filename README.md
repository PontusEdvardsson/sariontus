# Sari & Ontus Wedding Page

A simple static wedding landing page inspired by the olive monogram design in the reference screenshot.

## Files

- `index.html` - page content and sections
- `styles.css` - layout, typography, colors, and responsive styles

## Publish on GitHub Pages

1. Push this folder to the `sariontus` GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose the `main` branch and `/root`, then save.
